Sintetizador do DECOMP
=================================

**Versão:** |release|

**Data:** |today|

O *sintetizador-decomp* é um pacote Python para formatação dos arquivos de saída do modelo `DECOMP <https://www.cepel.br/linhas-de-pesquisa/decomp/>`_. O DECOMP é
desenvolvido pelo `CEPEL <http://www.cepel.br/>`_ e utilizado para a planejamento da operação do Sistema Interligado Nacional (SIN).
